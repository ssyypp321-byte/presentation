ADR-0001
플랫폼 선택 — Flutter
✓ Flutter React Native Unity Android Native
단일 코드베이스로 iOS/Android 동시 지원. JS 브릿지 성능 우려 없이 60fps 보장. Flame 게임 엔진 활용 가능.
ADR-0002
상태 관리 — Provider
✓ Provider setState BLoC Riverpod
게임 상태(체력·던전·인벤토리)를 독립적으로 분리. setState보다 안전하고 BLoC보다 보일러플레이트 적음.
ADR-0003
데이터 저장 — Hive (로컬 우선)
✓ Hive Firebase SQLite SharedPreferences
오프라인 플레이 보장. Firebase는 네트워크 의존으로 오프라인 불가. 빠른 NoSQL 구조로 게임 데이터에 적합.
